Build AI Medical Pre-Diagnosis Frontend
We will build the frontend structure for a modern AI medical pre-diagnosis web application using React (Vite), TailwindCSS, and Framer Motion, focusing on a premium SaaS-level design inspired by Apple Health and Stripe.



Proposed Changes
Setup and Configuration
* Create a new Vite + React project named frontend.
* Install dependencies: tailwindcss, postcss, autoprefixer, framer-motion, lucide-react (for clean, modern icons), and react-router-dom (for routing between the 3 pages).
* Configure Tailwind: Set up tailwind.config.js with custom theme settings (soft shadows, rounded borders, modern font, custom color palette including subtle blues).
* Global CSS: Update src/index.css to include Tailwind directives and some global styling for a clean, modern look.
